# slice_mini_app

Slice es una mini-aplicación nativa dentro del ecosistema de Lemon que revoluciona la gestión del dinero al permitir a los usuarios crear "reservas" de fondos. Esta funcionalidad actúa como una alcancía digital que segrega montos específicos de la cuenta principal para fines predeterminados.
Su característica distintiva es la capacidad multi-activo: cada reserva puede mantenerse en diferentes divisas, incluyendo pesos, dolares y criptomonedas variadas. Esto permite al usuario elegir el activo más adecuado para su meta de ahorro, protegiendo o potenciando su valor. El objetivo es fomentar el ahorro disciplinado con total control sobre la moneda de la reserva.
